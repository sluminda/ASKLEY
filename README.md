## 📖 Overview

**Askley** is a web-based AI chatbot that allows users to ask questions and receive intelligent responses through a clean and interactive browser interface.

The application is designed as a **self-hosted AI solution**, reducing dependency on paid external APIs while improving privacy, flexibility, and deployment control.

---

## 🌐 Live Website

🔗 **Online Hosting:** [https://askley.site](https://askley.site)

The application is already deployed and publicly accessible online.

---

## ✨ Key Features

- 💬 Interactive AI chatbot interface
- 🤖 AI-powered response generation using Ollama
- 🎨 Clean and modern frontend design
- 🧾 Markdown rendering with **marked.js**
- 🔄 New chat reset functionality
- ⚠️ Error handling for network and timeout issues
- 🔒 Secure live deployment with HTTPS
- ☁️ Cloudflare-connected domain setup
